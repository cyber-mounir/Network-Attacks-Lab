# Network-Attacks-Lab

## About This Project
This repository documents my learning journey in **network security**
through practical labs and simulations.

Each project demonstrates how a specific network attack works,
why it is effective, and how it can be mitigated.

---

## What You Will Find
- Hands-on network attack simulations
- Clear explanations and diagrams
- Defensive security techniques
- A final attack-chain scenario

---

## Skills Developed
- Network traffic analysis
- Understanding of Layer 2 & Layer 3 protocols
- Security mindset and ethical awareness
- Use of GNS3 and Linux-based tools
